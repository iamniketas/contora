# Contora v0.5.2 macOS Pilot

Pilot macOS build for Apple Silicon Macs. This build is ad-hoc signed, but it is not signed with an Apple Developer ID and is not notarized by Apple.

## Download

Use `Contora-macOS-0.5.2-macos-pilot-arm64-unsigned.dmg` for the simplest install flow.

The ZIP is provided as a fallback.

## Install

1. Download the DMG.
2. Open it and drag `Contora.app` to `Applications`.
3. Open `Contora.app`.
4. If macOS blocks the first launch, open `System Settings -> Privacy & Security`, click `Open Anyway` for Contora, then confirm `Open`.
5. Grant Microphone and Screen Recording permissions when macOS asks. Restart Contora after granting permissions if macOS asks you to.

## Local Whisper

1. Open Contora.
2. In the main window, use `Backend -> Set Up Local Whisper`.
3. Wait for setup to finish. The pilot build bundles the local runtime and can download the Whisper model from the app.
4. Record audio and use local transcription. Diarization is available through the bundled local Whisper runtime.

If setup gets into a bad state, use `Backend -> Repair -> Repair Runtime`. To start over completely, use `Backend -> Repair -> Delete Runtime and Models`.

## Known Limitations

- macOS will warn on first launch because this pilot is not Developer ID signed or notarized.
- Apple Silicon only (`arm64`).
- The first model download can be large and slow.
- This is a pilot release, not a polished public distribution.
